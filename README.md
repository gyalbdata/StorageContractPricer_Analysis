# Storage Contract Pricer Analysis

An interactive dashboard and analysis tool designed to evaluate natural gas storage contracts, estimate profitability, and visualize key pricing metrics.

## Live Dashboard

🔗 **View the Dashboard:**
https://gyalbdata.github.io/StorageContractPricer_Analysis/

## Project Overview

This project analyzes storage contract opportunities by examining market prices, storage costs, injection and withdrawal schedules, and potential profit scenarios. The dashboard provides visual insights to help users understand the financial performance of storage strategies.

## Features

* Interactive data visualizations
* Storage contract pricing analysis
* Profitability calculations
* Market trend exploration
* User-friendly dashboard interface

## Technologies Used
